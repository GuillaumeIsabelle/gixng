# GIX Ng Libraries

## Install
```sh
npm i gixng --save
npm i gixng --save-dev

yarn add gixng --save
yarn add gixng --dev
```

## Usage
```javascript
//Variables
var { outputPath, projectName } = require('gixng');

```
```javascript
var { outputPath, projectName,angularConfig } = require('gixng');

```
```javascript
var { angularConfig } = require('gixng');

```
